Ejercicios propuestos ordinario parcial

Ejercicio

Calcular las siguientes integrales indefinidas utilizando una substitución apropiada.

  1. $\displaystyle{\int x(x^{2} +1)^{5}dx}$.
  2. $\displaystyle{\int 4x e^{x^{2}}dx}$.
  3. $\displaystyle{\int x^{2} \sqrt{x^{3}+1} dx}$.
  4. $\displaystyle{\int \frac{(\mathrm{ln}(x))^{2}}{ x} dx}$.
  5. $\displaystyle{\int x^{2}e^{x^{3}}dx}$.
  6. $\displaystyle{\int \frac{2-x}{\sqrt{2x^{2} - 8x +1}}dx}$.
  7. $\displaystyle{\int \frac{x}{x^{2} +1}dx}$.
  8. $\displaystyle{\int \mathrm{tan}(x)dx}$.
  9. $\displaystyle{\int (x^{2} + 2x + 3)^{6}(x+1) dx}$.
  10. $\displaystyle{\int x e^{-x^{2}}dx}$.
  11. $\displaystyle{\int \frac{(1+\mathrm{ln}(x))^{3}}{x} dx}$.
  12. $\displaystyle{\int (x^{6} -6x)^{7}(x^{2} -2) dx}$.
  13. $\displaystyle{\int \frac{\mathrm{ln}(2x)}{x} dx}$.
  14. $\displaystyle{\int \frac{x^{4}}{x^{5} +1}dx}$.
  15. $\displaystyle{\int \frac{x-3}{(1- 6x +x^{2})^{2} }dx}$.
  16. $\displaystyle{\int \frac{\mathrm{ln}(\sqrt{x})}{x} dx}$.
  17. $\displaystyle{\int \frac{x^{2}- 2x}{x^{3} -3x^{2} +1} dx}$.
  18. $\displaystyle{\int \frac{\sqrt{\mathrm{ln}(x)}}{x} dx}$.
  19. $\displaystyle{\int \frac{e^{\sqrt{x}}}{\sqrt{x}} dx}$.
  20. $\displaystyle{\int x^{-2} \left( \frac{1}{x} +2 \right)^{2}dx}$.
  21. $\displaystyle{\int \frac{x^{2}}{1- x^{5}}dx}$.
  22. $\displaystyle{\int \frac{8x}{e^{x^{2}}} dx}$.
  23. $\displaystyle{\int \frac{1}{x \mathrm{ln}(x^{2})}dx}$.
  24. $\displaystyle{\int e^{x} (1 + e^{x} )^{5} dx}$.
  25. $\displaystyle{\int \frac{e^{x}}{1 + 2e^{x}} dx}$.
  26. $\displaystyle{\int \frac{e^{-x}}{1 -e^{-x}} dx}$.
  27. $\displaystyle{\int \frac{2}{x \left(\mathrm{ln}(x)\right)^{4}}dx}$.
  28. $\displaystyle{\int \frac{dx}{3 -5x}}$.
  29. $\displaystyle{\int \frac{e^{x} + e^{-x} }{e^{x} - e^{-x}} dx}$.
  30. $\displaystyle{\int \frac{(1 + e^{-x})^{3} }{e^{x}} dx}$.
